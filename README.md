# **SubParallel**

**SubParallel** 是一个基于 Python Tkinter 开发的轻量级桌面工具，专为**双语 ASS 字幕文件**设计。它能够将原文（英文）与译文（中文）**并排显示**，支持同步滚动、智能分离提取以及文本搜索。

## **✨ 核心功能**

- **🔄 双语并排对照**
  - 自动解析 ASS 文件中的 `Dialogue` 行。
  - 智能识别“译文字幕”和“英文字幕”样式。
  - 将中英文字幕在左右两个面板中**逐行对齐**展示，方便对比阅读。
- **📜 智能文本分离**
  - 内置正则表达式引擎，自动去除 ASS 特效标签（如 `{...}`）。
  - 智能处理换行符 (`\\N`)，自动区分并分离混合在一起的中英文段落。
  - 支持多种文件编码自动检测 (UTF-8, GBK, GB2312, Latin-1)，解决乱码问题。
- **⚡ 交互式体验**
  - **同步滚动**：滚动左侧原文时，右侧译文自动跟随滚动，保持视线同步。
  - **灵活视图**：支持“左右分栏”与“原文译文结合”两种视图模式一键切换。
  - **全局搜索**：支持在当前视图中高亮搜索关键词，并快速跳转定位。
  - **字体调节**：支持 `Ctrl + 鼠标滚轮` 实时调整字体大小，保护视力。
- **💾 数据导出**
  - 支持将提取后的纯文本**原文**或**译文**单独保存为 `.txt` 文件，便于后续处理或导入其他软件。

## **🛠️ 技术栈**

- **语言**: Python 3.x
- **GUI 框架**: Tkinter (ttk)
- **核心库**: `re` (正则表达式), `os`, `tkinter.filedialog`, `tkinter.messagebox`
- **无外部依赖**: 无需安装任何第三方库，开箱即用。

## **🚀 快速开始**

### **1. 环境准备**

确保您的计算机已安装 Python 3.6 或更高版本。

```
python --version
```

### **2. 运行程序**

下载本项目代码后，直接在终端运行主脚本：

```
python subtitle_display.py  # 假设您的文件名为 subtitle_display.py
```

*注：请将文件名替换为您实际保存的文件名。*

### **3. 使用流程**

1. 点击 **“选择ASS文件”** 按钮，加载您的双语字幕文件。
2. 点击 **“处理文件”** (或按 `F5`)，程序将自动解析并并在左右窗口显示中英文内容。
3. 使用滚动条或鼠标滚轮同步浏览字幕。
4. 如需保存，点击 **“保存原文”** 或 **“保存译文”**。

## **💡 工作原理**

本工具通过正则表达式解析 ASS 文件的 `Dialogue` 行：

1. **提取样式与文本**：识别 `Style` 字段（如 "译文字幕", "英文字幕"）和 `Text` 字段。
2. **清洗数据**：移除 ASS 特有的特效代码 `{...}`。
3. **智能分离**：对于包含 `\\N` 的双语行，通过检测中文字符范围 (`\u4e00-\u9fff`) 自动判断哪部分是中文，哪部分是英文，并将其拆分到对应的列表。
4. **同步渲染**：利用 Tkinter 的事件绑定机制，实现两个 Text 组件的滚动位置实时同步。
